1

Handling Missing Values

Do not drop rows with missing values because missingness is spread across most columns. Impute categorical variables (HomePlanet, Destination, CryoSleep, VIP) with the mode, and numerical variables (Age, RoomService, FoodCourt, ShoppingMall, Spa, VRDeck) with the median.

2

Feature Engineering & Binning

Use domain-based Age bins: Child (0-12), Teenager (13-17), Young Adult (18-35), Adult (36-55), Senior (55+). Create Total_Spent by summing RoomService, FoodCourt, ShoppingMall, Spa, and VRDeck.

3

Outlier Detection & Treatment

Use IQR or boxplots to detect outliers in spending columns. Prefer capping or log transformation over row deletion because high spenders are likely valid observations, not data-entry errors.

4

Normalization / Scaling

Scale numerical variables so large spending ranges do not dominate model learning (for example, Age 0-79 versus FoodCourt 0-29,813). Use Standardization (Z-score) or Min-Max scaling.
